PicServe

PicServe™ 는 단일 원본 이미지로 실시간 리사이징·크롭·압축·포맷 변환을 제공하는 이미지 서빙 솔루션입니다. 웹과 앱 전반에서 더 빠른 로딩, 더 낮은 전송 비용을 동시에 달성합니다.

  • URL 파라미터 기반 실시간 리사이징/크롭
  • WebP 자동 변환으로 전송량 절감
    파일 크기 감소: JPEG 대비 약 25-35% 더 작은 용량 로딩 속도 향상: 네트워크 대역폭 절약 및 페이지 로딩 시간 단축 고품질 유지: 손실 압축과 무손실 압축 모두 지원 투명도 지원: PNG처럼 알파 채널 지원
  • 지능형 압축과 스마트 크롭으로 품질 유지
  • 워터마크 삽입으로 저작권 보호
    • 텍스트 워터마크: 저작권 표시나 브랜드명을 텍스트로 삽입
    • 로고 워터마크: 투명 배경의 로고 이미지를 삽입 (300x300 PNG)
    • 위치 및 투명도 조절: 우측 하단에 35% 투명도로 배치 (코드 수정으로 변경 가능)
    • 자동 크기 조정: 이미지 크기에 맞춰 워터마크 폰트 크기 자동 조절
  • 백업/삭제 예약과 디스크 사용 통계 제공
    • 데이터 삭제 예약: 오래된 이미지 파일을 자동으로 정리하여 저장 공간 확보
    • 데이터 백업: 중요한 이미지를 주기적으로 백업하여 데이터 손실 방지
    • 유연한 스케줄링: 분/시간/일/월/년 단위로 세밀한 예약 설정 가능
  • 썸네일 생성 시 인물의 얼굴을 중심점으로 자르기
    • 자동 얼굴 감지: 분류기를 통해 정면 얼굴을 정확하게 인식
    • 스마트 크롭: 감지된 얼굴을 중심으로 최적의 영역을 자동으로 잘라냄
    • 오탐지 방지: 최소 5번 이상 중복 검출된 영역만 유효한 얼굴로 인정
    • 경계 처리: 이미지 경계를 벗어나지 않도록 자동 보정

통합 대시보드에서 정책을 일괄 관리하고, 이미지 정리·백업 스케줄링과 보안 인증을 함께 제공합니다.
