Vision Intelligence

UniBox™는 ML용 라벨링·어노테이션 관리와 인공신경망(Artificial Neural Network, ANN) 모델을 이용한 학습/추론 솔루션입니다.

상세보기 도입 문의

실시간 객체 탐지 및
컴퓨터 비전 모델 기반

이미지 어노테이션부터 모델 학습, 추론, 결과 검토까지를 하나의 UI 흐름으로 연결한 데이터-학습 통합 워크플로우 솔루션입니다.

  • 데이터 준비 속도 향상: 다중 이미지 업로드 및 디렉토리 기반 이미지 로딩 지원.
  • YOLOv8 중심 운영 일관성: YOLOv8 탐지 결과(Bounding Box, 라벨, confidence) 시각화.
  • 라벨링과 어노테이션: 도형 기반(사각형/원형/폴리곤) 어노테이션으로 다양한 객체 형상 대응

탐지 결과(Bounding Box, 라벨, confidence) 시각화와 선택한 탐지 박스의 좌표/크기/점수 상세 확인 가능
